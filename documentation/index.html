Instalation

  • After unzip the download pack, you'll found a Template Folder with all the files.

  • You can view this Template in any browser, you can display or edit the Template without an internet connection.(May not wotrk fonts).

  • Now go to your cpanel or open your FTP Client (like Filezilla) and upload the content of the Template on your server root.

  • Once the files are done uploading go to www.yourdomainname.com/index.html

CSS Files

These are the css files that are loaded into templates in Head Section.
In template using 6 CSS files. The file style.css contains general styling, such as anchor tag colors, font-sizes, etc.

If you want to customize some parts of the css, it's recommended to use a custom css file style.css where you put all your customizations. This will make it much more easier if you need to update the template some day.
The style.css already includes all font selectors & rules. So if you want to change the font, just replace the font name with yours.

 

Fonts

All fonts (except icon fonts) which are used by this template are comming from the Google font library.
You can of course change/use other fonts. The style.css file has a full list with all elements using which font.

Javascript Libraries

All js scripts are inlcuded at the bottom of the html file just before the body closes.

Most of the animation in this site is carried out from the customs scripts.
In addition to the custom scripts, I implement a few "tried and true" plugins to create the effects.
This plugin is packed, so you won't need to manually edit anything in the file.

jQuery and Bootstrap is a Javascript librarys that greatly reduces the amount of code that you must write.
